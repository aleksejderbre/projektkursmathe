Was ist eine Gleichung?

Eine Gleichung ist eine mathematische Aussage, in der zwei Ausdrücke mit einem Gleichheitszeichen verbunden sind. Beispiel: 3x + 2 = 8

Wie löst man eine Gleichung?

Ziel ist es, die unbekannte Zahl (meist x) zu finden. Dafür gibt es einige einfache Schritte:

  • Schritt 1: Die Gleichung vereinfachen (falls nötig), z. B. Klammern auflösen oder ähnliche Terme zusammenfassen.
  • Schritt 2: Zahlen und Variablen trennen: Alle Zahlen auf eine Seite bringen, alle x-Terme auf die andere.
  • Schritt 3: Die Unbekannte isolieren, indem man mit der Umkehroperation arbeitet.

Beispiele

  • Beispiel 1:
    3x + 2 = 8
    → 3x = 8 - 2 (2 subtrahieren)
    → 3x = 6
    → x = 6 / 3 (durch 3 teilen)
    → x = 2
  • Beispiel 2:
    5x - 7 = 18
    → 5x = 18 + 7 (7 addieren)
    → 5x = 25
    → x = 25 / 5 (durch 5 teilen)
    → x = 5
