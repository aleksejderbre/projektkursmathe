Was sind quadratische Gleichungen?

Quadratische Gleichungen sind Gleichungen, in denen die Unbekannte (meistens "x") in der zweiten Potenz vorkommt. Die allgemeine Form ist:

ax² + bx + c = 0

Dabei sind a, b und c Zahlen (Koeffizienten), und a darf nicht 0 sein.

Warum beschäftigen wir uns damit?

  • Viele Anwendungen: Quadratische Gleichungen beschreiben viele reale Situationen, z.B. bei der Berechnung von Flächen, Flugbahnen oder Wachstums- und Zerfallsprozessen.
  • Grundlegendes Verständnis: Das Lösen quadratischer Gleichungen ist wichtig für viele weitere mathematische Themen.

Lösungswege

Es gibt verschiedene Wege, um quadratische Gleichungen zu lösen:

1. Faktorisieren

Versuche, die Gleichung in der Form (px + q)(rx + s) = 0 zu schreiben. Dann sind die Lösungen x = -q/p oder x = -s/r.

Beispiel: x² + 5x + 6 = (x + 2)(x + 3) = 0. Lösungen: x = -2 oder x = -3.

2. Mitternachtsformel (abc-Formel oder quadratische Lösungsformel)

Die Mitternachtsformel ist eine allgemeine Formel zur Berechnung der Lösungen:

x = (-b ± √(b² - 4ac)) / 2a

Beispiel: x² + 2x - 3 = 0. a = 1, b = 2, c = -3.
x = (-2 ± √(2² - 4 * 1 * -3)) / 2 * 1 = (-2 ± √16) / 2.
Lösungen: x = 1 oder x = -3.

3. Quadratische Ergänzung

Durch quadratische Ergänzung kann die Gleichung in eine Form gebracht werden, aus der die Lösungen direkt abgelesen werden können.

Beispiele

  • Beispiel 1 (Faktorisieren):

    x² - 4x = 0
    x(x - 4) = 0
    Lösungen: x = 0 oder x = 4

  • Beispiel 2 (Mitternachtsformel):

    2x² + 5x - 3 = 0
    x = (-5 ± √(5² - 4 * 2 * -3)) / 2 * 2
    Lösungen: x = 0,5 oder x = -3
